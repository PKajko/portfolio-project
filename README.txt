Personal Portfolio Website
==========================

Overveiw
--------
This is my personal portfolio website project as part of my Codecademy Full-Stack Web Development course.
It gives a breif summarry of myself, my learning thus far and a little about myself personally.
Below that I have listed a few examples of previous projects I have worked on with links to GitHub and some of the technologies I have learned.

Features
--------
* Responsive layout using Flexbox and media queries.
* Interactive profile image (click to toggle shape).
* Projects section with links to GitHub repositories.
* Clean, simple design focused on readability.
* Contact page for reaching out.

Technologies Used
-----------------
* HTML5
* CSS
* JavaScript
* Git and GitHub for version control

Contact
-------
If you would like to get in contact, you can use the contact page by following the link at the bottom of the main page labelled "Contact Details".
